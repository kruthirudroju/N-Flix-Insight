# N-Flix-Insight

A premium, high-accuracy movie and TV show data analytics and content recommendation engine built using Python and Streamlit. This application applies an explicit weighted attribute scoring strategy via machine learning text vectorization to surface hyper-relevant cinematic matches, wrapped completely in a sleek, custom-styled minimalist black and crimson user interface.

## 🚀 Key Features

* **Advanced Matching Architecture:** Evaluates content using an isolated text vectorization pipeline. Matches are linearly weighted to guarantee precision (**40% Genres**, **40% Directors**, and **20% Main Cast**).
* **Minimalist Black & Crimson UI:** Custom-engineered frontend utilizing injected CSS layouts, crisp type scales, edge-to-edge fluid responsive positioning, and deep charcoal workspace containers.
* **Blank State Initialization:** Protects background system memory by ensuring computation matrices only execute upon direct user search query baseline allocation.
* **Global Distribution Telemetry:** Live data visualization tracking longitudinal studio release curves, asset volume splits, and top global origin production market metrics.

---

## 🛠️ System Architecture & Stack

* **Frontend Engine:** Streamlit (Injected Custom CSS Layouts)
* **Data Processing:** Pandas
* **Data Visualizations:** Plotly Express & Plotly Graph Objects
* **Vectorization & Distance Scoring:** Scikit-Learn (TF-IDF Vectorizer & Cosine Similarity)
